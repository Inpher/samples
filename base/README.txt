Requirements:
================
* the latest version of Java (JDK 1.8)
* maven

please run:
================
* bin/setup.sh to configure the sdk and/or install solr locally
* bin/gui.sh to test the framework with the GUI application

Getting started:
================
the inpher-samples folder contains the source code of minimal programs to
test the library.
It is a maven project, which can be imported
directly into eclipse.
