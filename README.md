# gui-sample-app
This is a GUI sample application to show developers how to use the Inpher SDK

### Requirements

- Docker
- OS X or Linux
- JAVA: JDK 8

### Build

To build the App before running it, run:
```
    ./gradlew installApp
```

### How to start

To start the GUI and all the backends:
```
    chmod u+x run.sh
    ./run.sh
```
